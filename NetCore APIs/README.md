# Intro

This folder contains material about .Net Core APIs.

To create a project in VS Code type:
```dotnet new webapi -o MyFirstApi```

To generate a self-signed cert for localhost:
```dotnet dev-certs https --trust```

To execute the API:
```dotnet run```
